Version: 1.0

# Objective

Create a world-class GitHub profile that reflects the engineering maturity of a Technical Lead and Platform Engineer rather than a traditional resume.

The profile should immediately communicate expertise in:

- Backend Engineering
- Distributed Systems
- Platform Engineering
- Cloud Native Infrastructure
- Workflow Orchestration
- AI Infrastructure
- Developer Productivity

The overall feeling should resemble the GitHub profiles of senior engineers working at companies such as Stripe, Cloudflare, Datadog, Netflix, Anthropic, OpenAI, Vercel or Uber.

The profile should feel like a polished engineering portfolio rather than a résumé.

---

# Design Philosophy

The profile must prioritize:

- readability
- simplicity
- professionalism
- engineering credibility
- storytelling

Avoid:

- resume dumps
- excessive badges
- random widgets
- flashy animations
- emoji overload
- technology buzzword stuffing

Everything should reinforce one story:

> "This engineer designs platforms."
